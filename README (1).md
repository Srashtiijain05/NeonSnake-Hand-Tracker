# 🐍 NeonSnake Hand Tracker

> Real-time hand tracking with glowing snake trails, gesture detection, particle effects, and a cinematic neon starfield — powered by MediaPipe & OpenCV.

---

## ✨ Features

- 🐍 **Snake trails** — 5 glowing worms with forked tongues follow each fingertip
- 🎭 **Gesture detection** — Fist, Peace, Open, Thumbs Up, OK, Rock On
- 🌊 **Ripple effects** — Expanding neon rings on every gesture
- 🎵 **Gesture sounds** — Unique synth tones via pygame
- 🌌 **Starfield background** — Animated twinkling stars
- 🎨 **Paint mode** — Draw glowing trails on screen (`P` to toggle)
- 💥 **Particle explosions** — Speed-reactive bursts from fingertips
- 📸 **Screenshot** — Save canvas anytime (`S`)

---

## 🚀 Quick Start

```bash
pip install mediapipe opencv-python numpy sounddevice pygame
python neon_hand_tracker_v2.py
```

---

## 🎮 Controls

| Key | Action |
|-----|--------|
| `Q` | Quit |
| `P` | Toggle paint mode |
| `C` | Clear paint canvas |
| `S` | Save screenshot |

---

## 🖐 Gestures

| Gesture | Effect |
|---------|--------|
| ✊ Fist | Explosion + orange ripple |
| ✌️ Peace | Fireworks + cyan ripple |
| 🖐 Open hand | Rainbow burst |
| 👍 Thumbs Up | Gold shower |
| 👌 OK | Spiral color burst |
| 🤘 Rock On | Red shockwave |
